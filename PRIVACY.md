# Privacy Policy

### Tasks for Canvas (Grey-Out Edition) does not collect, transmit, sell, or share any personal data.

* All functionality runs locally in the user’s browser.

* No analytics, tracking, or advertising is used.

* No data is sent to external servers.

* The extension uses browser storage only to save user preferences (such as enabling or disabling the grey-out feature).

If you choose to contact the developer (for example via email or GitHub Issues), any information you provide is used solely to respond to your inquiry.

**Contact:** support is provided on a best-effort basis via the project’s GitHub repository.
